Thanks for trying, but the original Hyper Text Coffee Pot Control Protocol (HTCPCP) hasn't any native teapot extensions available. In order to successfully view this webpage, please make sure that your connected teapot and Internet client or browser support the RFC 7168 HTCPCP extension for Tea Efflux Appliances (TEA). If not, please upgrade to a newer teapot. Thanks for visiting, anyways!
